# FEMsa Website

Welcome to the FEMsa (Fes Medical Student Association) Website project! This web application serves as a central hub for FEMsa, providing information and resources for medical students in Fes.

## Features

- **Homepage:** Engaging landing page with highlights and announcements.
- **About Us:** Learn about FEMsa's mission, history, and leadership.
- **Comitee:** Get to know the students
- **Contact:** Connect with FEMsa through our contact form.

## Technologies Used

- **Frontend:** HTML, CSS, JavaScript
- **Backend:** JavaScript
- **Deployment:** Github
